1. HTML
    2. Primary Use Cases Key features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS
    3. Primary Use Cases Key features/Benefits
    CSS is used for styling and layout control of web pages.
    • Selectively styles HTML elements using selectors and properties.
    • Allows for responsive design with media queries.
    • Enables precise control over typography, colors, and spacing.

  3. JavaScript
    4. Primary Use Cases Key features/Benefits
    JavaScript is used for adding interactivity and behavior to web pages.
    • Handles user interactions and events through event listeners.
    • Dynamically manipulates the Document Object Model (DOM) for real-time updates.
    • Supports asynchronous programming for non-blocking tasks and AJAX requests
